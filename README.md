# Portfolio Website - CSS Mastery

## Project Description

This project is a fully responsive personal portfolio website built using HTML5 and CSS3. The website showcases my profile, skills, projects, and contact information while demonstrating modern CSS layout techniques including Flexbox and CSS Grid.

The project was developed as part of the IYF Weekend Academy CSS Mastery assignment.

---

# Features

* Responsive mobile-first design
* Flexbox navigation and layouts
* CSS Grid photo/project gallery
* Google Fonts typography system
* Smooth hover animations and transitions
* Clean and modern user interface
* Multi-page portfolio structure
